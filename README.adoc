# Compile {#compile}

## Stand-alone

As a prerequisite, you have to install
[Feel++](http://www.feelpp.org/docs/develop/BuildingP.html) on your system.

then do the following
~~~~~{.sh}
git clone https://github.com/feelpp/quickstart
mkdir build
cd build
cmake ../quickstart
~~~~~
